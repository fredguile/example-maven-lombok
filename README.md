# example-maven-lombok

A dummy example Maven + Lombok project that works in Neovim. 

Can be used to troubleshoot Neovim Java JSP (`nvim-java`).

See [installation instructions](https://projectlombok.org/setup/maven), it may be required to run this command on new projects:

```sh 
java -jar lombok.jar createMavenECJBootstrap -o /path/to/project/root
```
